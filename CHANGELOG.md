## [0.0.1] - 2025-11-24

### Added
- Testing infrastructure: pytest + pyfakefs + coverage configuration
- CI/CD: GitHub Actions workflows for testing and linting
- Pre-commit hooks: black, ruff, mypy, yaml validation
- Test scaffolding: conftest.py with fixtures and test structure
- Development documentation: CONTRIBUTING.md with setup guide
- Coverage reporting: HTML, XML, and terminal output
- Test markers: safety, integration, slow, requires_filesystem

### Changed
- Version changed from 0.1.0 to 0.0.1 (pre-alpha development)
